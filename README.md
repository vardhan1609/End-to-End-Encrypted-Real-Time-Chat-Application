# End-to-End Encrypted Chat Application

A secure real-time chat application supporting private and group conversations with room creation and invite-based joining. The system uses end-to-end encryption to ensure complete message confidentiality, where encryption and decryption occur only on the client side.

## Features
- Create and join chat rooms using invite codes
- Private and group messaging support
- End-to-end encrypted communication
- Real-time messaging using WebSockets
- Secure authentication and room management

## Tech Stack
- Frontend: Vite + JavaScript
- Backend: Python (FastAPI)
- Real-time: WebSockets (Socket.IO)
- Encryption: Modern cryptographic protocols (E2EE)
- Database: PostgreSQL, Redis
